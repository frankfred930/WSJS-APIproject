# Finnish Commuter Traffic App
A live commuter traffic web app using the Digitraffic API(https://rata.digitraffic.fi/api).  
Displays live train timetables and info for selected stations in Finland.

## Technologies Used
- HTML
- CSS
- Vanilla JavaScript (Fetch API)
- Digitraffic Open Data API

## Live Demo
View on Netlify (https://your-netlify-link.netlify.app)

## How It Works
1. Fetches list of Finnish train stations dynamically.
2. Lets the user select or search for a station.
3. Displays live arriving and departing train info.
4. Styled with CSS and built using only native JS (no libraries).
